## Pre-Visit Research Checklist

Before visiting a restaurant to pitch:

- [ ] Find on Google Maps (reviews, photos, hours)
- [ ] Find on TheFork (menu, prices)
- [ ] Find on TripAdvisor (reviews)
- [ ] Find their Instagram (vibe, photos)
- [ ] Note their Google rating
- [ ] Identify owner/manager name
- [ ] Identify if they have staff who answer questions
- [ ] Note any existing QR codes or tech
- [ ] Prepare pitch: "I'm building an AI concierge..."
- [ ] Prepare demo on phone/tablet

## Pitch Script (30 seconds)

```
Hi, I'm [name]. I'm building an AI concierge for restaurants.
Customers scan a QR code on the table and can ask the AI anything about the menu —
"What wine pairs with the octopus?", "Is there a vegetarian option?", 
"What time does the kitchen close?"

It works like a smart waiter who knows the entire menu.

I have a working demo here. Can I show you?

[SHOW DEMO]

If I set this up for you, it's €299 for the install and first month. After that it's €49/month. Would that be interesting if it works well for your tables?
```

## Demo Flow

```
1. Show AI answering 3 questions:
   - "What's your signature dish?"
   - "What wine goes with the lamb?"
   - "Do you have vegetarian options?"

2. Show the voice interface (speak, not type)

3. Show subtitles

4. Then ask:
   "If I install this for you, it's €299 including setup and your first month live. After that it's €49/month. Does that feel reasonable if the demo works?"

5. If yes -> "Great. Let me send you the setup link and book the install. Can I get your email and WhatsApp?"
```
