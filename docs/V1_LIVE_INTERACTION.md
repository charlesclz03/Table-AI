# Gustia Live Interaction

Last updated: 2026-04-05

## What Is Live Today

### Guest Experience

- QR-to-chat flow
- language selection
- theme selection
- real-time AI replies
- voice plus text interaction
- OpenAI voice playback with browser fallback
- subtitles
- demo CTA after three guest messages

### Owner Experience

- owner auth
- pricing-auth-checkout flow
- admin dashboard
- menu editor
- concierge training workspace
- quiz editor
- QR poster generation
- billing overview
- analytics dashboard
- invite claiming flow
- owner changelog feed

## What Is Still Not Live

- waiter handoff
- POS integration
- order forwarding
- owner-visible live conversation replay

## Key Clarification

Earlier Gustia docs described owner analytics and dashboard visibility as later phases.

That is no longer true.

Owner-facing admin, billing, analytics, and release visibility already ship in the current app.
