## Common Commands

- install deps: `npm install`
- start dev server: `npm run dev`
- onboarding suggestions: `npm run onboarding:suggest`
- lint: `npm run lint`
- type-check: `npm run type-check`
- tests: `npm run test`
- e2e tests: `npm run test:e2e`
- production build: `npm run build`
