# Gustia UI Design Language

Last updated: 2026-04-05

## Core Visual Direction

- one shared azulejos backdrop behind the interface
- glassmorphism panels, chips, inputs, and tinted CTAs
- mobile-first composition
- the wine sphere is the hero visual inside guest onboarding and chat

## Wine Sphere

The sphere should feel:

- glassy
- liquid-filled
- softly glowing
- responsive to speaking state

Motion language:

- idle pulse
- speaking slosh and wave motion
- shimmer sweep
- theme-specific glow tone

## Active Guest Themes

The current guest selector exposes four active themes:

- Red Wine
- White Wine
- Rose
- Green Wine

The broader theme library still includes Champagne, but it is not currently part of the active guest selector.

## Typography and Tone

- premium but readable
- short, owner-friendly marketing copy
- guest-facing copy should feel confident and hospitable
